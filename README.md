# banking-queue

## Tech Stack:
* 1. Spring Boot - 2.1.3
* 2. MySQL - 5.7.21-0ubuntu0.16.04.1
* 3. Java -1.8

## Code setup
### Build
* Install mvn 3.3 or above
* Install Java 8 or above
* run in terminal 
```
mvn clean install
```
* Deploy code at localhost:8080 using terminal
```
mvn spring-boot:run
```
